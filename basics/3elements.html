HTML Elements

An HTML Elements is usually composed of a Start Tag Elements Content  and an End Tag

For Example: <p>Start Tag   Element Content   End Tag</p>

Nested HTML Element

There are some cases that an HTML element can contain one or more HTML elements.

For Example: <p> <i> Italicized Text </i> </p>

Empty Elements

Empty Elements are elements that do not have an element content and an end tag.

For Example: <hr /> , <br /> , <meta /> , <img /> , <input /> , <link /> etc.
