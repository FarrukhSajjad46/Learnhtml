HTML Attributes

HTML Attributes used to add more information to an HTML element.

Important Things To Remember

  1. Html attributes are found in HTML Tags.
  2. Html attributes only appears at start tag.It will never be appear in end tag.
  3. Html elements can have multiple attributes.
  4. Html elements are composed of name / value pairs.
  5. There are some attributes that can be used on all HTML element though they may have not effects on some elements.Thay are called Global Attiributes.For example: Contenteditable
    • It can have two possible value.
    • True: indicates element should be editable.
    • False: indicates that element should not be editable.
    • The cotenteditable attribute does not require a value if it is present at the start tag then it will become editable otherwise not.

An HTML attribute is composed of :

  • an attribute name
  • an equal = sign
  • a value surrounded by quotation marks "value"
  • It looks like this: attributename="value".

HTML Attributes With Examples

We use the attribute to define the language of an HTML file.For Example: <html lang="en-US">

Attribute href=

Links are defined using the anchor < a > element.

On the example above we used the href attribute to tell the browser where to go.

When clicked the user will be redirected to http://www.example.com.

Example: <a href="http://www.example.com">Go to http://www.example.com</a>

Attribute title=

The title attribute provide a tooltip for HTML element.

Example: <a href="#link" title="I serve as a tooltip">Link </a>

Attribute style=

On the example given above we have created a paragraph using the <p> element.

We also used the style: attribute to chang its font-size and color.

Example: <p style="font-size: 40px; color: "gold"> I am a paragraph </p>

Attribute id= and class=

The id: and class: attributes give references to elements inside an html document

Multiple elements can have the same class values/names.

The id values should be unique for each element.

These help us select elements in style sheets and scripts.

Examples: <div id="name"> some content goes here </div> <div class="name"> some content goes here </div>
