HTML Introduction

HTML stands for HyperText Markup Language.

HTML describe the structure of web pages.

Structure is given below,

            
                <!DOCTYPE html>
                 <html lang="en">
                    <head>
                         <title>Introduction</title>
                          <meta name="viewport" content="width=device-width" initial-scale="1.0"/>
                    <head/>
                      <body>
                            <h1>HTML Introduction</h1>
                            <p>Welcome to HTML.</p>
                      </body>
                 </html>

HTML Elements Details

<!DOCTYPE html>
Defines the types of the document, Which is HTML5.
<html>
It encloses all the tags of document, for example: Elements , Style Sheets , Scripts , Multimedia and lot more.
<head>
This element encloses all the metadata of a document, Which will not be displayed on the main content of Web Page.This could include style sheets, scipts, <title>, <meta> and lot more.
<title>
This element defines the title of Web Page. It appears on the upper part of the browser.
<body>
This element encloses element like <h1> to <h6>, <img />, <b>, <i> and lot more.
<h1>
This element defines heading.
<p>
This element defines paragraph.
